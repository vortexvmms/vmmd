# VMMS — Vortex Manpower Management System

Internal web application for daily manpower allocation, attendance,
WhatsApp updates and monthly man-hour reporting.

**Spec:** VMMS-SPEC-001 Rev 5 · **Status:** Phase 1 (project setup)

## Structure

| Folder | What it is |
|---|---|
| `frontend/` | The website (PWA) — published free on GitHub Pages |
| `backend/`  | The server (Python FastAPI) — runs free on Render.com |
| `db/`       | Database scripts for Supabase (added in Phase 2) |
| `docs/`     | Guides and documentation |
| `.github/`  | Automatic publishing of the frontend |

## Setup

Follow **docs/VMMS Setup Guide.docx** — written step-by-step,
no technical knowledge needed.
